# Puzzle generator game

## Key points

- Family focused
- Asynchronous
- Has to work on mobile phones
- Drawing component
- Iterative
- Score keeping
- Limited tries
- Multiple participants
- Asymetric gameplay
- Team building
- Ice breaker

## Brain storm

- Drawing hidden behind a grid of squares. Each square is solved with a puzzle
- 3x3 or 2x2 grid

## Description

- A person given the title puzzle maker, the rest are puzzle guessers
- A person is given a concept: sustantive + adjective + location
- Such person makes a drawing with a generous timer (2-3 min)
- The puzzle maker takes a mini quiz with 9 (or 4) questions
- Questions are about their preferences with 4 alternatives (Favorite pokemon, Favorite movie, Favorite electrodomestic)
    - They have to be silly in nature
    - Questions with same alternatives shouldn't repeat for each puzzle maker on repeat sessions
- A 9x9 grid is shown to puzzle guessers upon pressing each square a question from the previous quiz is shown
    - Each column represents a different set of parts of the original concept
- Puzzle guessers can guess each question once per hour (2 hours?)
- Questions may be ai-generated or set up by the a team
- When a puzzle guesser asserts a correct question:
    - a square of their choice is removed, revealing a part of the puzzle maker drawing
    - a word is added to their dictionary
- To win a puzzle guesser needs to select 3 words from the dictionary which corresponds with the original concept given to the puzzle maker
    - the dictionary is built guessing questions
    - The dictionary consists on 3 correct concepts and 6 wrong concepts
    - Wrong concepts need to be adjacent to the real concept, but not synonynms
